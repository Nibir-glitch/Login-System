# Readme file
This is a login system project

Features: 
## It doesn't allow two users to have same username. The second user must use different username to signup
## Same goes for email. Each user must have unique email.
## A password recovery system is also implemented if the user forgets his/her password in any case. For that case, the user's username and email must match. This feature will be further updated for more security as its security is very low.
